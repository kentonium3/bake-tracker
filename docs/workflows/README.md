# Workflows Directory

This directory contains process documentation for development and testing workflows.

## Workflow Documents

### testing_workflow.md
- **Purpose:** Data management workflow for testing and development cycles
- **Audience:** Developers and testers
- **Topics:**
  - Simple workflow (no schema changes)
  - Export/import workflow (with schema changes)
  - Database backup and restore
  - Version upgrade procedures

## Purpose of Workflow Docs

Workflow documents provide:
1. Step-by-step procedures for common tasks
2. Best practices for data management
3. Testing protocols
4. Troubleshooting guides

These are **procedural guides** - they explain **how to do things** rather than **what needs to be built**.
